## Context

XML format confirmed: TraceEvent elements with attributes (sequence, ruleNumber, stepMethod, stepPage, step, stepStatus, eventType, elapsed, name, inskey, keyname, rsname, rsvers) and child elements (Sequence, DateTime, EventType, EventName, Interaction, ThreadName, WorkPool, Message). Tree built by matching Begin/End eventType pairs using a stack. Flamegraph uses sequence numbers as x-axis since elapsed is sparse.
